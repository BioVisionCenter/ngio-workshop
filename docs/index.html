ngio Workshop

Hands-on notebooks for learning the basics of ngio.

Running the Notebooks

The notebooks are self-contained and manage their own dependencies. You only need uv installed.

Clone the repository:

git clone https://github.com/fractal-analytics-platform/ngio-workshop.git
cd ngio-workshop

Run a notebook interactively:

uvx marimo run --sandbox notebooks/1_ngio_basics.py

Or open it for editing:

uvx marimo edit --sandbox notebooks/1_ngio_basics.py
